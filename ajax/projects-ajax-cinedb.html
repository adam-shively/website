cineDB

Project Info:

A movie web application that consumes the TMDB API. Movies can be imported by adding them to a Postgresql database. Imported movies can be added to custom movie collections created by an administrative user. All users can obtain administrative access by clicking "Demo Admin" in the nav bar.
