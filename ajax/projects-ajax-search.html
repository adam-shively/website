Search Party

Project Info:

This is a word and text file searcher that searches for a user entered phrase. User can enter a directory or word/text file path. If a directory is entered, all sub directories and selected file types will be searched. User can chose to search through word documents, text files, or both with file searches that were successful being displayed. Any directories or files that caused an error are also displayed. User can chose to search for files with an exact phrase, for files with any word in that phrase, or for files containing all words in that phrase that appear anywhere in the document.
